# tapxpense_website

Static marketing site for TapXpense.

## Structure

- `index.html` — landing page
- `privacy.html` — privacy policy
- `terms.html` — terms of service
- `css/styles.css` — shared styles
- `js/main.js` — landing page interactions
- `assets/` — screenshots and logo (add before deploy)

Open `index.html` locally or deploy as static files (e.g. GitHub Pages, Vercel).
